## Sobre o projeto 🖥️
A proposta do projeto é criar uma página de cadastro escolar, utilizando as **ferramentas padrão de desenvolvimento web (HTML, CSS e JavaScript)**. O projeto será constamente atualizado, com novas funções, interações e páginas.

## O que está sendo trabalhado 💡

Proposto pelo professor da disciplina de Desenvolvimento Web, o projeto visa alcançar o entendimento completo da criação de uma página web completa, com estilo e interações. 

## Como rodar esse projeto ⌨️

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com](https://github.com/luiskkjk/desenvolvimento-web)
    ```
2.  **Entre na pasta:**
    ```bash
    cd desenvolvimento-web
    ```
3. **Instale a extensão Live Server**
    
4.  **Inicie o projeto:**
   Clicando com botão direito em "index.html" e o iniciando com o Live Server
